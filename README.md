# AI Lead Intelligence CRM

An AI-powered CRM application that helps businesses manage leads and detect suspicious domains.

## Features
- Lead management dashboard
- AI-based lead quality scoring
- Suspicious domain detection
- Lead pipeline tracking
- CSV export functionality

## Tech Stack
- React
- Node.js
- Supabase (Database)
- AI Analysis

## Live Demo
https://lead-wise-ai-crm-453964.withmattr.app

## Project Overview
This project helps businesses manage and analyze leads. The system assigns a quality score to each lead and flags suspicious websites based on domain patterns.
